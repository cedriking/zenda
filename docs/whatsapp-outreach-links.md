# WhatsApp Direct Links — One-Click Outreach

These links open WhatsApp Web/Desktop with the first-contact message pre-filled.
Just click the link, fill in the business name, and hit send.

## How to Use

1. Click a link below
2. Replace `[NOMBRE]` with the business name (e.g., "Salón Bella")
3. Replace `[TU_NOMBRE]` with your name
4. Send the message

### Quick Instructions for Finding Phone Numbers

1. Google Maps → search "salón de belleza [neighborhood]"
2. Click business → scroll to phone number
3. Format: country code + number (e.g., Mexico: `525512345678`)
4. Build link: `https://wa.me/525512345678?text=...`
