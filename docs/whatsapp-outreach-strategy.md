## Why pivot from email?

| Channel | Sent | Delivered | Replies | Conversion |
|---------|------|-----------|---------|------------|
| Cold email | 65+ | ~30 | 0 | 0% |
| WhatsApp direct | 0 | - | - | - |

LATAM SMB owners live on WhatsApp. They don't check email from unknown senders.
Zenda IS a WhatsApp product — we should use WhatsApp to sell it.

## Outreach Script (Spanish)

### Opening message
```
Hola! Soy Diego, creador de Zenda. Hice un asistente virtual por WhatsApp que agenda citas automáticamente para negocios como el tuyo.

¿Te molesta si te envío un enlace para que lo pruebes gratis? Es literalmente 1 minuto.
```

### If they respond positively
```
Excelente! Entra aquí y te activo el plan gratis:
https://zenda.bot/signup?coupon=aRgf7NZC

O si prefieres, puedo configurarlo por ti. Solo necesito:
1. Los servicios que ofreces
2. Tu horario
3. Los nombres de tus estilistas/profesionales
```

### If they ask "how much"
```
Hay plan gratis para hasta 25 clientes mensuales. Si necesitas más, los planes empiezan en $29 USD/mes con 50% de descuento para founders.
```

### If no response after 24h
```
Entiendo si no es el momento. Si algún día quieres automatizar tus citas por WhatsApp, Zenda está disponible gratis en:
https://zenda.bot/signup?coupon=aRgf7NZC

¡Éxito con [negocio]!
```

## How to find WhatsApp numbers

1. Google: `[business name] WhatsApp` or `[business name] telefono`
2. Google Business Profile — most LATAM businesses list their WhatsApp
3. Facebook page — "Enviar mensaje" button usually goes to WhatsApp
4. Instagram — "Contact" button often shows WhatsApp number

## How to message

1. Save number in phone contacts
2. Open WhatsApp, find contact, send script
3. OR use wa.me link: `https://wa.me/5215512345678?text=Hola...`
