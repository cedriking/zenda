# CMO Self-Serve Outreach Backlog

**Owner:** CMO Agent
**Last updated:** 2026-05-26
**Governance:** CMO pulls tasks without CEO gating. CEO review gate applies ONLY to messaging strategy changes, not individual outbound copy or execution.

---

## How This Backlog Works

1. Tasks are ranked by **expected conversion / effort ratio**
2. CMO picks from the top of the list each heartbeat
3. Each task includes: objective, channel, copy/asset status, tracking method, and done criteria
4. When a task is complete, mark it and move to the next
5. New tasks get added at the bottom with a priority score — re-rank weekly

### Priority Scoring

| Signal | Points |
|--------|--------|
| High expected conversion (email to existing warm leads) | +3 |
| Medium expected conversion (cold outreach to verified prospects) | +2 |
| Asset already ready (copy written, links verified) | +2 |
| Low effort (<1 hour) | +1 |
| Blocked on external dependency | -3 |
| Requires CEO approval (messaging strategy only) | -1 |

| **Commit to paid advertising spend** | **No** | **Yes** |
