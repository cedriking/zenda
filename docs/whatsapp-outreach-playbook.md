## Step 1: Find Prospects

### Google Maps Search
For each niche, search Google Maps:

```
barbería [ciudad] WhatsApp
dentista [ciudad] WhatsApp
taller mecánico [ciudad] WhatsApp
```

**Target cities** (by population + WhatsApp adoption):
- CDMX, Guadalajara, Monterrey, Puebla, Querétaro, Mérida
- Bogotá, Medellín, Cali
- Lima, Arequipa
- Buenos Aires, Córdoba

**What to look for:**
1. Business has a WhatsApp number listed on Google Maps
2. 4+ stars rating (established business)
3. 50+ reviews (they care about reputation)
4. Has a website (tech-forward)

### Google Search Queries
```
site:facebook.com "barbería" "WhatsApp" "agendar cita" México
site:instagram.com "barbería" "WhatsApp" "citas"
"agendar cita por WhatsApp" barbería
```

### Social Media
Search Instagram/Facebook for business profiles that:
- Post regularly (active business)
- Have WhatsApp button on profile
- Are appointment-based (not walk-in only)

## Step 4: Daily Execution Plan

| Time | Activity | Target |
|------|----------|--------|

**Expected conversion:** 5-10% reply rate → 2% sign-up rate = 4 sign-ups/week

**Day 0 (first contact):** Send Template A/B/C
**Day 2:** If no reply → "Hola, solo quería asegurarme de que vieras mi mensaje. Si no es el momento, sin problema 😊"
**Day 7:** If no reply → Final message: "Último mensaje, lo prometo 😄 Si alguna vez necesitas automatizar tus citas por WhatsApp, aquí estamos: https://zenda.bot/es/demo"

## Cost Estimate

- WhatsApp Business: Free
- Time investment: 3-4 hours/day
- Expected CAC: $0 (time only) vs. $50-100/lead via Google Ads

## Key Metrics to Watch

| Metric | Target | Red Flag |
|--------|--------|----------|
| Reply rate | >5% | <2% (adjust template) |
| Demo booking rate | >2% | <1% (improve targeting) |
| Sign-up rate | >1% | <0.5% (review pricing page) |
| Time to first reply | <24h | >48h (send at better times) |

## Notes

- Best times to message: 10am-12pm and 4pm-6pm (local time)
- Always message from a business WhatsApp account, not personal
- Include the demo link in EVERY message
- Track everything — data drives optimization
