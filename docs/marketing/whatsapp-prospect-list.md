# WhatsApp Outreach Prospect List — Zenda

## How to Use This List

1. Open the business name in Google Maps
2. Click the WhatsApp button (if available) to start a chat
3. Use the appropriate opener script from `docs/marketing/whatsapp-outreach-playbook.md`
4. Track results in the spreadsheet template (same doc)

If no WhatsApp button on Google Maps, try the business's Facebook page or Instagram — most LATAM businesses list their WhatsApp number there.

## Execution Plan

**Week 1 (Days 1-7):**
- Contact Priority 1 Mexico (20 businesses) — 3-4/day
- Use Mexico-specific opener scripts from playbook
- Follow up next day with non-responders

**Week 2 (Days 8-14):**
- Contact Priority 2 Argentina (17 businesses) — 3/day
- Follow up with Mexico non-responders
- Add Google Maps prospects as discovered

**Week 3 (Days 15-21):**
- Contact Priority 3 Colombia (7 businesses)
- Contact Priority 4 Chile/Peru (6 businesses)
- Follow up all non-responders

**Week 4 (Days 22-30):**
- Second follow-up round for all non-responders
- Focus on hot leads from weeks 1-3
- Expand with fresh Google Maps prospects

**Target:** 50 businesses contacted in 30 days → 15-25 responses → 5-8 demos → 2-4 signups
